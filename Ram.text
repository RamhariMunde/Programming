Programming Language:

A programming language is a system of notation
for writing computer programs. Most programming 
languages are text-based formal languages, but 
they may also be graphical. They are a kind of 
computer language.

What is C ?
The C programming language is a procedural and
general-purpose language that provides low-level
access to system memory. A program written in
C must be run through a C compiler to convert 
it into an executable that a computer can run.
